# WIFI 暴力破解

> 利用 py_wifi 加上字典跑wifi密码

项目修改自github开源项目，修改后支持中英文ssid，成功率取决于字典和密码难度。

### 操作步骤

1. 安装 python3.7
2. 双击或者命令行运行 py_wifi.py
3. 搜索附近WIFI
4. 选择字典
5. 双击列表中某个SSID 选中WIFI账号
6. 开始破解
